Дизайн сверстан строго по макету по 2-ум screen (1920px, 320px).
В проекте находится версия для разработки. Проект собран при помощи webpack. Для просмотра development версии необходимо установить все модули(npm install) и запустить команду npm run start. Запустится webpack-dev-server. Deploy версия находится по адресу https://sergeymasalski.github.io/Deploy_Test/

1. Дизайн сверстан строго по макету по 2-ум screen (1920px, 320px).
2. В проекте находится версия для разработки. Проект собран при помощи webpack. Для просмотра development версии необходимо установить все модули(npm install) и запустить команду npm run start. Запустится webpack-dev-server. Deploy версия находится по адресу https://sergeymasalski.github.io/Deploy_Test/
3. Для версии для телефонов 320px присутствует функция бургер-меню. При нажатии на кнопку EN или картинку планеты происходит изменение текста.
